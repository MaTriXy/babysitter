# Survey Design Agent

Design and analyze user surveys for UX research.

## Overview

This agent creates effective user surveys and analyzes results, ensuring valid quantitative insights for UX decision-making.

## Key Capabilities

- **Question Design**: Optimized wording and scales
- **Flow Logic**: Skip logic and branching
- **Bias Detection**: Response bias mitigation
- **Statistical Analysis**: SUS, NPS, CSAT calculations

## When to Use

- Creating user satisfaction surveys
- Analyzing survey results
- Designing usability questionnaires
- Conducting preference research
