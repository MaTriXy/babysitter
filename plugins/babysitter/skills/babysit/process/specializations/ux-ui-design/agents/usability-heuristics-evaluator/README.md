# Usability Heuristics Evaluator Agent

Conduct heuristic evaluations based on Nielsen's usability heuristics.

## Overview

This agent performs systematic usability evaluations using Nielsen's 10 heuristics, identifying issues and providing prioritized recommendations.

## Key Capabilities

- **Heuristic Assessment**: All 10 Nielsen heuristics
- **Severity Rating**: 0-4 scale classification
- **Issue Documentation**: Context and screenshots
- **Competitive Analysis**: Benchmark comparisons

## When to Use

- Expert usability reviews
- Design critique sessions
- Competitive UX analysis
- Pre-launch usability checks
