# Template Customization Agent

Intelligent agent for creating and maintaining language-specific SDK templates.

## Purpose

This agent ensures SDK code generation produces idiomatic, high-quality code that feels native to each target language.

## Key Responsibilities

- Creating language-idiomatic templates
- Maintaining template compatibility
- Reviewing generated code quality
- Applying post-generation transformations

## When to Engage

- Setting up new language targets for code generation
- Updating templates for new API features
- Improving generated code quality
- Troubleshooting generation issues
