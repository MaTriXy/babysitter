# Rasa NLU Integration

Configures Rasa NLU pipelines for intent classification and entity extraction.

## Overview

This skill provides Rasa NLU integration, enabling the configuration and training of intent classification and entity extraction pipelines.

## Key Features

- Configurable NLU pipelines
- Intent classification
- Entity extraction
- DIETClassifier support
- Fallback handling

## Usage

Use this skill when building conversational AI systems using the Rasa framework.
