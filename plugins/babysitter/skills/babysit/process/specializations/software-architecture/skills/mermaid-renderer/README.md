# Mermaid Diagram Renderer

Renders Mermaid diagrams to image formats with theme customization and Markdown integration support.

## Features

- Multiple output formats (PNG, SVG, PDF)
- Theme support (default, dark, forest, neutral)
- All Mermaid diagram types
- Markdown embedding support

## Usage

Provide Mermaid definition and output format. Integrates seamlessly with documentation workflows.
