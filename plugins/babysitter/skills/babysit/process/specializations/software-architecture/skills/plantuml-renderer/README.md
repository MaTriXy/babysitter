# PlantUML Renderer

Renders PlantUML diagrams to image formats (PNG, SVG, PDF) with theme support and batch processing capabilities.

## Features

- Multiple output formats (PNG, SVG, PDF, EPS)
- Custom themes and skinparams
- Batch rendering support
- All diagram types (sequence, class, component, deployment, activity, state)

## Usage

Provide PlantUML source and desired output format. Supports both inline source and file paths.
