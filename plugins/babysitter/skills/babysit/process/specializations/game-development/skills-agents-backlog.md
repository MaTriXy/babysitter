# Game Development - Skills and Agents Backlog

## Overview

Skills and agents identified to support Game Development processes. This backlog captures specialized capabilities that go beyond general-purpose agents to provide deep expertise in game engines, graphics programming, game AI, multiplayer networking, audio systems, and game design methodologies used in modern game production.
