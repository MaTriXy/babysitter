# LLM Classifier

Implements LLM-based zero-shot and few-shot classification for flexible intent detection.

## Overview

This skill provides LLM-based classification capabilities, enabling flexible intent detection without traditional model training.

## Key Features

- Zero-shot classification
- Few-shot learning
- Structured output
- Confidence scoring
- Multi-label support

## Usage

Use this skill when building intent classification systems that need flexibility and quick iteration without model training.
