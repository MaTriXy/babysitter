# REPL Development Skill

Expert skill for building interactive REPLs with rich editing and evaluation features.

## Quick Reference

- **Category**: Tooling
- **ID**: SK-021

## Key Capabilities

- Readline integration with history
- Multi-line input detection
- Tab completion for identifiers
- Incremental compilation for REPL
- Top-level expression evaluation
- Persistent REPL state
- Value pretty-printing
- Special REPL commands (:help, :type, etc.)

## When to Use

Use this skill when implementing:
- Interactive language REPLs
- Language exploration tools
- Interactive debuggers
- Educational programming environments
