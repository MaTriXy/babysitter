# HuggingFace Classifier

Fine-tunes and deploys HuggingFace transformer models for intent classification.

## Overview

This skill provides HuggingFace model fine-tuning and inference capabilities for building custom intent classifiers.

## Key Features

- Multiple base model options
- Trainer API integration
- Evaluation metrics
- Inference optimization
- Model deployment

## Usage

Use this skill when building custom intent classifiers that require fine-tuned transformer models.
