# Prototype Testing Agent

Facilitate and analyze prototype usability testing.

## Overview

This agent analyzes prototype usability testing data, measuring task completion, identifying pain points, and providing recommendations for design improvements.

## Key Capabilities

- **Task Metrics**: Completion rate, time, errors
- **Click Path Analysis**: Navigation pattern tracking
- **Pain Point Detection**: Usability issue identification
- **A/B Comparison**: Variant testing analysis

## When to Use

- Validating prototype designs
- Comparing design alternatives
- Identifying usability issues
- Measuring design iterations
